# `web-standards-ru-calendar-scripts`

Prints future events in a specified city

## Install

```sh
$ yarn install
```

## Usage

```sh
$ git submodule update --init && (cd calendar && git pull origin master)
$ yarn moscow
$ yarn spb
$ yarn print <russian-name>
```
